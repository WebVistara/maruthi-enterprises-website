# 🏭 Maruthi Enterprises Website

A modern, responsive multi-page website developed for **Maruthi Enterprises** to present their industrial services, portfolio, and company information.

---

## 🚀 Features

* 🌐 Multi-page website (Home, About, Services, Clients, Gallery, Contact)
* 📱 Fully responsive design for mobile and desktop
* ⚡ Fast loading with optimized images (`.webp`, `.avif`)
* 🏢 Clean and professional industrial UI
* 📷 Gallery section to showcase work
* 📞 Contact page for customer interaction

---

## 📂 Project Structure

```
Maruthi_Enterprises/
```
